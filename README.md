DigiTools – Premium Digital Product Marketplace
DigiTools is a modern, high-performance digital marketplace built with React and Vite. It allows users to browse premium digital assets, manage a dynamic shopping cart, and enjoy a seamless, professional UI/UX experience.

🚀 Features
🖼️ Dynamic Product Gallery
- Curated list of premium tools
- Real-time updates from a JSON data source
🛒 Advanced Shopping Cart
- Add/Remove items dynamically
- Real-time cart count in the Navbar
- Automatic total price calculation
🔄 Tab-Based Navigation
- Toggle between Products and Shopping Cart views
- Smooth navigation without page reloads
🔔 Interactive Notifications
- Integrated with React-Toastify
- Professional success and error alerts
📱 Fully Responsive Design
- Custom layouts for Desktop, Tablet, and Mobile
- Optimized specifically for 576px breakpoints
🎨 Clean Aesthetic
- Professional White & Purple theme
- High-quality studio-style imagery

✨ Why DigiTools?
- Fast loading and smooth navigation
- Professional UI/UX with modern design principles
- Responsive layouts for all devices
- User-friendly shopping experience

🛠️ Tech Stack
- React (UI framework)
- Vite (fast build tool)
- React-Toastify (notifications)
- CSS/Responsive Design (custom layouts)
