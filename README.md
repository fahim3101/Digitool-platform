# DigiTools - Premium Digital Product Marketplace

DigiTools is a modern, high-performance digital marketplace built with **React** and **Vite**. This platform allows users to browse premium digital assets, manage a dynamic shopping cart, and experience a seamless, professional UI/UX.

## 🚀 Key Features

- **Dynamic Product Gallery**: Displays a curated list of tools with real-time data from a JSON source.
- **Advanced Shopping Cart**: 
  - Add/Remove items dynamically.
  - Real-time cart count updates in the Navbar.
  - Automatic total price calculation.
- **Tab-Based Navigation**: Toggle between the "Products" view and the "Shopping Cart" without page reloads.
- **Interactive Notifications**: Integrated with **React-Toastify** for professional success and error alerts.
- **Fully Responsive**: Custom-built layouts for Desktop, Tablet, and Mobile (specifically optimized for **576px** breakpoints).
- **Clean Aesthetic**: Utilizing a professional "White & Purple" theme with high-quality studio-style imagery.
